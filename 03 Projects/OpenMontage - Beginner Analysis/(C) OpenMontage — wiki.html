00:00 · THE THESIS

Intelligence in Markdown & YAML. Code only for the deterministic work.

The single most transferable idea here — and why this repo matters to a software developer, quite apart from video.

Every prior corpus subject that "orchestrates" bakes its logic into code. OpenMontage does the opposite: the agent reads a YAML pipeline manifest → reads a Markdown "stage-director" skill → calls a Python tool → checkpoints JSON state → advances. Python does only what LLMs can't: deterministic tool execution, schema validation, state persistence, cost arithmetic. The creative and policy logic lives in readable, version-controllable text. That is your own LLM-Wiki pattern, generalized to industrial scale — which is exactly why its patterns steal cleanly into your Claude Code workflow.

00:03 · THE DISCIPLINE

The quality machinery is the gold

A dual-layer model — readable governance in Markdown/YAML, thin Python only for schema + state. Every mechanic here transfers to a software team.

reviewer.md

Scored review meta-skill

CHAI findings — Accurate (cite the exact field) · Complete (pattern-scan the class) · Constructive (propose a fix). A 6-dimension "slideshow-risk" score with a fail ≥ 4.0 threshold.

decision_log.schema.json

Decision audit trail

16 decision categories; every material choice logs options_considered (with scores + why-rejected), the selection, a reason, and a confidence. ADRs, for agents.

checkpoint-protocol.md

Checkpoint & resume

JSON state with in_progress + partial_progress markers — a long run resumes without redoing work, and gates on human approval at creative stages.

framework-smoke.yaml

Cheap CI contract harness

A 2-stage, no-tool-execution pipeline that validates manifest/schema/checkpoint contracts in CI without spending on renders. Catches breakage early.
