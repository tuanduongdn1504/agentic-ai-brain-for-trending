## 4. Outside-scope classification

### Storm Bear scope definition

Storm Bear vault focuses on:
- **Agent frameworks** — tools that orchestrate LLM agents
- **Agent methodologies** — ways to work with agents
- **Agent services + platforms** — infrastructure for agent deployment
- **Agent education** — learning resources for agents
- **Agent bridges** — connectors between agents and external services
- **Agent applications** — standalone agent products

**fish-speech is NONE of these.** It's a foundation model (TTS) that could be *used by* an agent, but is not itself an agent artifact.

### 5-tier taxonomy applicability

| Tier | Applicable to fish-speech? |
|------|---------------------------|
| T1 Assistant | ❌ — not an assistant framework |
| T2 Service | ❌ — not a multi-tenant platform |
| T3 Education | ❌ — not a learning resource |
| T4 Bridge | ❌ — not a bridge between agents and services |
| T5 Application | ❌ — not a standalone agent app |
| **Outside-scope** | **✅ — foundation model, not agent** |

> "Frameworks in agent-adjacent space exhibit 4 distinct intellectual-lineage archetypes:
> 1. Individual-author — cite specific person(s) as primary influence
> 2. Methodology — build on shared idea system (SDD, TDD, BMAD)
> 3. Community-viral no-lineage — emerge from community demand without cited predecessors
> 4. Research-paper-chain — build on network of cited academic/research projects
>
> Archetype distribution varies by space: agent-frameworks skew toward (1-3); foundation-models skew toward (4)."

### Prediction
