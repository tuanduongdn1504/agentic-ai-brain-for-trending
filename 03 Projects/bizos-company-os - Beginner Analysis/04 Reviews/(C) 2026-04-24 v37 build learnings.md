Bizos appears to be built WITH Claude Code (inferred from 4 signals). The corpus is ABOUT Claude Code. So bizos is a recursive artifact: a subject built with the tools the subject corpus documents. Meta-observation: first corpus wiki where subject-vs-tools relationship has this recursive character. Pattern #57 Recursive Corpus Reference (confirmed v27) is different kind of recursion (corpus wiki cites prior corpus wikis). Bizos has new recursion: subject-built-with-corpus-subjects-methodology. If more AI-built products get wiki'd, consider meta-pattern.

**Observation D — Cross-tier VN observation tracking gap:**

**Observation E — Pattern #29 absent-LICENSE ambiguity:**

But is it a license-category (my framing) or the absence of a license-category (alternative framing)? Treating as license-category because: (1) it's a deliberate choice (Alex Le could have added LICENSE); (2) it has consistent semantic ("all rights reserved by default"); (3) it's explicitly documented in README (proprietary statement).
