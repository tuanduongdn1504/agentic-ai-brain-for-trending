## Implications

1. **README is well-structured marketing collateral; underlying course is 21% built then abandoned** — classic pattern of content-marketing repos where SEO/star-fishing motivates initial build, then maintenance cost defeats commercial-funnel ROI
2. **Affiliate-link integration as declared quality standard** — affiliate revenue is part of the product, not a side-effect; this is honest commercially but creates perverse incentive (recommend tools that pay best, not tools that work best)
3. **License claim "as-is for educational purposes" is informal** — not an OSI license name; default copyright applies; commercial reuse legally ambiguous
4. **Disclaimer disclosure is generic, not per-link** — borderline FTC compliance (varies by jurisdiction)
5. **Heavy Firebase Storage CDN dependency for images** — if Firebase project deleted, all README images break (single point of brand-collapse failure)
