## What it is

A **VitePress-based vibe-coding curriculum** organized by Chinese AI/ML open-source community **Datawhalechina** with project lead **Sanbu** and Tsinghua University contributors. 4-stage progressive curriculum (Stage 0 "kindergarten" + Stage 1-3) covering 120+ interactive topics across 13 languages, deployable to 5 targets (Vercel + GitHub Pages + Microsoft Azure + Docker + nginx), with a **CORPUS-FIRST `llms.txt`** AI-Agent-optimized tutorial index.

The corpus-defining novelty: **CORPUS-FIRST explicit PRO-VIBE positioning** — names "vibe coding 2026" as methodology + treats traditional learning order as inverted ("Build first, understand after") + endorses dialogue-with-AI as primary development mode.
