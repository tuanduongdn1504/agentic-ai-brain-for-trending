### easy-vibe là gì?

easy-vibe là một khóa học miễn phí (CC BY-NC-SA 4.0) về **vibe coding 2026** — phương pháp lập trình thời đại AI nơi bạn nói chuyện với AI để xây dựng ứng dụng. Khóa học được tổ chức bởi **Datawhalechina** (cộng đồng AI/ML mã nguồn mở Trung Quốc) cùng với cố vấn học thuật **Fang Ke** (Đại học Thanh Hoa) và các sinh viên đóng góp.

> "Nhảy vào và vibe cùng nhau — nếu bạn có thể nói chuyện, bạn có thể xây dựng app."

## 🇬🇧 English

### What is easy-vibe?

easy-vibe is a free (CC BY-NC-SA 4.0) curriculum for **vibe coding 2026** — the AI-era programming methodology where you build apps by talking to AI tools. The course is organized by **Datawhalechina** (Chinese AI/ML open-source community) with academic mentor **Fang Ke** (Tsinghua University) and student contributors.

> "Jump right in and vibe together — if you can talk, you can build apps."

### The inverted approach

| Traditional coding | easy-vibe (vibe coding) |
|---------------------|-------------------------|
| Syntax → theory → build later | **Build first → understand mechanisms after** |
| Single-tool mastery | Multi-tool dialogue fluency |
| Code-as-product | **Conversation-as-product** |
| Deep technical onboarding | Conversational onboarding |

### 4-stage curriculum

| Stage | Audience | Outcome |
|-------|----------|---------|
| **Stage 0 ("Kindergarten")** | Absolute beginners | First conversational interactions with AI coding tools |
| **Stage 1 — Beginner Entry** | PMs / Founders | Idea validation + prototyping + AI capability integration |
| **Stage 2 — Full-Stack** | Junior developers | Frontend + backend APIs + databases + deployment + payment |
| **Stage 3 — Advanced** | Senior developers | Claude Code + MCP + Skills + Agent Teams |

### Supported AI tools

- **Claude Code** (MCP + Skills + Agent Teams) — Stage 3 anchor
- **Cursor** — Stage 1-2 prototyping
- **Trae** — Chinese-localized AI IDE
- **VS Code AI extensions** — accessibility tier
- **Dify** — knowledge base integration

### How to start

1. Visit https://datawhalechina.github.io/easy-vibe/welcome.html
2. Choose your stage based on your role
3. Absolute beginner → start at Stage 0
4. Already worked with AI tools → skip to Stage 1+
5. AI Agent reader (Claude / GPT) → read `llms.txt` for intelligent routing

### When NOT to use easy-vibe

- You need certificate-issuing formal education (this is community-curated)
- You require Python ML/data-science focus (curriculum is JavaScript-dominant 89.1%)
- You need commercial-license teaching materials (CC BY-NC-SA blocks commercial use)
- You prefer book-anchored sequential reading (this is AI-navigable curriculum)

### What makes easy-vibe distinctive

- 🌟 **13,202 stars** at 144 days = 91.7 stars/day
- 🌐 **13 languages** (corpus-record i18n)
- 📚 **120+ interactive topics** + 9-knowledge-domain appendix
- 🏗️ **Datawhalechina** + **Sanbu** (lead) + **Fang Ke** (Tsinghua) + 5+ Tsinghua students
- 📄 **`llms.txt`** — file designed for AI Agent consumption (decision tree + dual-layer index)
