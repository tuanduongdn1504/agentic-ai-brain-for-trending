## Distinctions (the anti-generalize-to-fit checks)

- **OpenHands v30** — same tier (T5 Agent-as-application) but the **autonomous SWE-agent platform** flavor (sandboxed/batch/SDK/cloud). Kilo Code is the **interactive editor-embedded pair-programmer** flavor. Different product shape → the corpus distinguishes finer than this elsewhere (session-multiplexer vs orchestration-platform vs GUI-client are three separate entries).
- **AutoGPT** — general autonomous agent, not IDE-embedded coding.
- **DeepSeek-TUI v72** — terminal client, single-vendor; not a multi-model IDE platform.
- **PilotDeck v175** §C — self-contained agent OS organized around per-project WorkSpaces (desktop-task productivity); not an IDE-embedded coding agent.
- **CodePilot v161** §C — a desktop **GUI that wraps** a CLI coding agent (Claude Code); Kilo Code is its own agent, editor-embedded, not a wrapper around another agent.
- **GLM-5 v176** — a raw model (the engine); Kilo Code is the harness that runs engines.
- **The tooling-around-agents surface** (cc-switch v73 / devspace v171 / claude-tap v173 / codebase-memory-mcp v172 / Agent-Reach v174) — they augment/bridge/inspect/feed an agent; Kilo Code is the agent.
