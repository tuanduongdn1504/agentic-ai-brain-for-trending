## What it is
A **human-driven multiplexer** for multi-agent coding. You spawn several AI coding agents at once — each on its own git branch via an automatically-created **git worktree**, each running in its own persistent **tmux session**, optionally inside a **Docker / Podman / Apple-Container sandbox** with configurable host-directory mounts. A terminal **TUI** (ratatui) and a browser **web dashboard** (React 19 / Vite / Tailwind v4 / xterm.js, installable as a **PWA**) let you watch every session's live status, view diffs, edit files, and approve/answer the agent. Sessions **survive** the TUI closing, SSH disconnects, and reboots (that's the tmux foundation). It explicitly **"does not replace agents like Claude Code or OpenCode; it orchestrates them"** — and the human is the orchestrator (it is *not* an agent-of-agents platform). Most useful at **3+ concurrent agents** — a deliberately scoped target.

## How it integrates (the instructive part)
- **tmux + git worktrees** — one isolated session per agent, one worktree per branch (auto-created + cleaned up); branch isolation without manual branch management; sessions persist across disconnect/reboot.
- **Docker / Podman / Apple-Container sandboxing** — container isolation with shared-auth volumes and *configurable* host-dir mounts (explicit control over what the container can touch).
- **Web dashboard = an opt-in compile feature** — `cargo build` ships the TUI-only binary; `cargo build --features serve` adds the dashboard/HTTP server.
- **Agent Client Protocol (ACP) structured view** — for ACP-capable agents, renders agent state natively (plan panels, tool-call cards, **swipe-to-approve**, mobile-first) instead of raw terminal bytes; any session can flip to **raw tmux passthrough** (the universal fallback). ACP is a **consumed external open standard** (Zed-ecosystem, crates.io `agent-client-protocol`); AoE's top-level `acp-worker/` is its ACP worker.
- **Remote / phone** — press **`R`** in the TUI to expose the dashboard over HTTPS with **QR + passphrase** auth via **Tailscale Funnel** or **Cloudflare Tunnel**; act on a blocked agent from your phone.
- **CLI + HTTP API** — drive sessions from the command line or **external orchestrators** ("integrates with tools like OpenClaw").
- **Distinctive engineering** — a single-source-of-truth config via a **`#[setting(...)]` Rust attribute macro** (one field auto-generates the TUI + web + server surfaces via schema derivation, "no hand-maintained allowlists"); server-projected theming (`Theme` → `ResolvedTheme`, a new theme is "a one-file drop"). Heavy CI for a ~5-month project: Codecov 75%-patch gate, dual Playwright suites, git-cliff changelog, weekly automated staging-PR releases.
- **Design** — ships a `DESIGN.md` aesthetic design-language doc ("Industrial Warmth," warm amber/copper, explicitly anti-"Linear-style-template," benchmarked vs Warp/Zed/Ghostty); namesake "Empire" theme = the Age-of-Empires pun is intentional. README credits **agent-deck (Go + Bubble Tea)** as inspiration.
