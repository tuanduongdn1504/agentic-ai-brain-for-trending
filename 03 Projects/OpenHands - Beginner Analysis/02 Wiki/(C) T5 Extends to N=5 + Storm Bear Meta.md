### Storm Bear caveats

- **No Vietnamese localization** in README or docs — system prompts need VN context manually
- **Docker-heavy** — 315 MB repo + runtime image may exceed VN client laptop specs
- **LLM API cost** — model-agnostic routing means user selects provider; sustained use accumulates cost
- **Commercial-tier clarity** — "free Cloud tier + paid enterprise" boundaries unclear from docs probe; need deeper evaluation before recommending to clients
- **Security-analysis** is production-grade but user-configured — operator must set policies; out-of-box defaults may not match VN client compliance
- **Benevolent dictator model** — continuity depends on 3-person core team; enterprise risk vs pure-OSS-community
