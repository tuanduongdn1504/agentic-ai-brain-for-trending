# Releases and Launches — Week 19 2026

> What the three sources say shipped, leaked, or was publicly demoed in the week ending 2026-05-13. Items grouped by lab/company. Cite-tagged from the source bundle.

---

## Anthropic

- **Claude Opus 4.7 — "leaked"** per WorldofAI's framing; presented as the next-gen flagship that an alleged Opus 4.6 quality dip would set up [Source 2].
- **Anthropic AI Studio** — reported as a "full-stack vibe coding environment" for building and deploying apps, positioned to compete with Replit [Source 2].
- **Claude Code update** — flagged by WorldofAI's headline but discussed mostly as a "token tax" controversy where extra tokens are allegedly injected into requests; the recommended bypass is pinning to `npx claude-code@2.1.98` [Source 2].
- **Claude Desktop app as "universal remote"** — Sisinty describes it as letting users plug in any model (local, OpenRouter, Foundry) while keeping the Claude interface [Source 1].
- **Claude "Live Artifacts"** — interactive, auto-refreshing dashboards that turn a chat session into a dynamic workspace [Source 1].

## OpenAI

- **GPT 5.5** — Sisinty calls it the "smartest and most intuitive model" OpenAI has ever shipped [Source 1]. WorldofAI references "a new GPT model" without confirming the exact 5.5 numbering [Source 2].
- **OpenAI Workspace agents** — agents that live inside Slack, Linear, etc. to monitor feedback and file tickets autonomously 24/7 [Source 1].
- **ChatGPT Images 2.0 / GPT Image Gen 2** — adds a "thinking" toggle where the model uses Python to calculate coordinates before generating or editing an image; better prompt adherence and cross-generation consistency [Source 1, Source 2].
- **OpenAI "Chronicle"** — opt-in tool that takes periodic screenshots of a user's computer to build a richer memory; flagged as a developer-community privacy flashpoint [Source 1].
- **Codeex** — terminal-style coding tool grouped alongside Claude Code by Ryan Dozer as the preferred non-web interface [Source 3].

## Google

- **"Stitch" tool** — now supports **design.md**, an open format that lets AI agents read brand rules (fonts, spacing, components) to produce consistent UIs [Source 1].

## Chinese labs

- **Kimmy K2.6** — Sisinty's headline subject: free model that he says "embarrassed" Claude and ChatGPT; can code for 12 hours straight across 4,000+ tool calls. Presented as "free open-source AI" without licensing caveats [Source 1].
- **MiniMax M2.7** — WorldofAI covers it skeptically: the company calls it fully open source, but the license restricts commercial use, so per his read it does not meet the official OSI definition [Source 2].
- **Gem Opus 426B** — fine-tuned model called out by WorldofAI specifically for "agent-style workflows" testing [Source 2]. (Source does not specify origin lab; treat as an experimental fine-tune mentioned in passing.)

## Tooling, infrastructure, partnerships

- **Cursor AI × SpaceX partnership** — Sisinty reports Cursor gained access to SpaceX-tier supercompute power, positioning it as the engineer's primary editor [Source 1].
- **MCP (Model Context Protocol)** — Dozer pitches it as the connective tissue for plugging AI agents into WordPress, Stripe, and social tools like Blato to build a "personal operating system" [Source 3].
- **"Skill stacks" / .md skill files** — Dozer's productized pattern: voice brain-dump → Markdown SOPs → sold as digital products or consulting upsells; he claims $3,000+ in passive income from it [Source 3].

## Security incidents and warnings

- **Vercel breach** — Sisinty cites a case where hackers used a simple AI tool to bypass multi-factor authentication and steal data from Vercel; framing is that the "new hack" is a "pretty AI app" requesting permissions [Source 1].
- **Workspace-permission hygiene** — Sisinty's tactical guidance: audit AI tools attached to Google Workspace and never click "Allow All" on new productivity apps [Source 1].

## Labor and ethics

- **Indian factory-worker camera footage for training** — WorldofAI flags this as a "concerning" trend: workers being made to wear cameras to record movements that will train AI replacing their physical labor [Source 2].

## Key Takeaways

- The week's biggest lab-level stories per these sources: Kimmy K2.6 beating Western models on benchmarks [Source 1], a Claude Opus 4.7 leak rumor tied to an Opus 4.6 quality-dip allegation [Source 2], and OpenAI's Workspace agents going from concept to in-app reality [Source 1].
- Anthropic's product surface is widening rapidly: Opus 4.7 rumor + AI Studio + Live Artifacts + Claude Desktop as a universal remote — the company is no longer just a model vendor in this week's framing [Source 1, Source 2].
- Chinese open(-ish) models dominate the "what shipped" share of voice; the licensing distinction (truly open vs. commercially restricted) is contested between Sources 1 and 2.
- Tool-side launches (Cursor × SpaceX, design.md in Google Stitch, MCP connectors) signal that the week's narrative center of gravity is moving from "new models" to "new substrates for agents" [Source 1, Source 3].
- The Vercel breach and Chronicle screenshotting are this week's headline security/privacy items — both in Source 1 only, which is a single-channel signal, not a confirmed cross-source story.
