## The 8 meta-skills (extraction)

1. **Process Interviewer** *(Ben-built)* — interviews you with 10–15+ questions to define a fuzzy/hard-to-articulate process **before** building a skill or complex task (anti "AI jumps to action too fast"). Outputs a confirmed summary → a full **PRD/brief** embedding Anthropic's skill-building best practices + your defined process. With a second brain, first checks the workspace to skip already-answered questions. **Adapted from Matt Pocock's "grill me" skill** ("Grail me skill from Matt Pocock, credits to him") + Anthropic best practices.
2. **Prompt Master** *(found online)* — turns a messy brain-dump (e.g. dictated via WhisperFlow) into an optimized **structured** prompt per best practices. Usage: append "optimize this prompt using the prompt master skill"; can pull second-brain context; can be set to **auto-run** via Cospace/general/folder instructions.
3. **Humanizer** — removes signs of AI-generated writing. **Explicitly based on Wikipedia's "Signs of AI writing" guide.** Run on anything AI-generated going public. Drops tells like "hope you're well", replaces "unlock"/vague hype with specifics. Customizable; chain as the last step of copywriting skills.
4. **Fact Checker** — systematic, evidence-based fact verification + misinformation ID; cross-references the web. 3 use cases: (a) check AI output, (b) check your **own** claims (e.g. a Notion video outline), (c) check **someone else's** article. Report: true / mostly-true / unverifiable / false counts. Chain into copywriting skills.
5. **Find Skills** — discovers other people's skills from the **skills.sh** ecosystem ("more than 91,000 skills" w/ install counts). Claude installs directly or gives a CLI command for Claude Code.
6. **Front-end Slides** — builds **animated HTML presentations** from scratch or by converting PPT/PowerPoint. Embeds slide-layout best practices. Style presets (defaults to "neon cyber" for AI topics); strongly brand-customizable (colors/fonts/design rules from a brand guide).
7. **Decision Toolkit** — make big decisions via systematic analysis + **first-principles** thinking. Principle: **"guide, don't decide."** Outputs a markdown decision framework **and** an interactive HTML wizard: frame the decision, "start fresh test", timing, who's affected, bias checks, opportunity cost, scenarios+probabilities, optional verdict, first-principle test.
8. **MCP Builder** *(built-in Anthropic skill — "many don't know about it")* — builds a custom **MCP server** for software that has an API but no MCP/connector (Ben built ones for **Circle.so** and **Reddit**). Reads the target's API docs, asks capability questions, outputs server + step-by-step + a **JSON snippet** to paste into the Claude config (Settings → Developer → Edit Config).

**Bonus skills mentioned briefly:** agent-browser skill (more efficient than Claude's native browser use); audio-transcriber skill (transcribe any audio/video on any site/media player); OpenRouter skill (test outputs across LLM providers — Gemini/OpenAI/etc.).
