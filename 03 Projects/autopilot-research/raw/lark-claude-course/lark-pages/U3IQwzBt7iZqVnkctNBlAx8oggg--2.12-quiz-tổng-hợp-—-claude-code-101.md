Giới thiệu​
Bạn đã đi qua 11 bài học — từ agentic loop đến hooks, từ /compact đến MCP, từ CLAUDE.md đến subagents. Quiz này là cơ hội để bạn tự kiểm tra mình đã nắm được bao nhiêu trước khi đưa Claude Code vào workflow thực tế.​
Quiz không phải "pass/fail". Nó là bản đồ — chỉ cho bạn biết mình đang đứng ở đâu và cần ôn lại gì.​
Cách làm:​
1.
Đọc từng câu hỏi, suy nghĩ đáp án trước khi nhìn đáp án​
2.
Ghi lại câu trả lời của bạn (A/B/C/D) ra giấy hoặc editor​
3.
Hoàn thành tất cả 20 câu, sau đó mới mở phần Đáp án​
4.
Với mỗi câu sai: đọc giải thích, click vào bài tham chiếu, ôn lại đúng section đó​
Cách tính điểm: 1 câu đúng = 1 điểm. Tổng 20 điểm. Xem bảng xếp hạng ở cuối bài.​
​

Câu 1​
Claude Code khác các AI chat thông thường (như Claude.ai) ở điểm nào cơ bản nhất?​
A) Claude Code dùng model mạnh hơn, nên giỏi code hơn​
B) Claude Code chạy trong một agentic loop — có thể tự dùng tools (bash, file edit, grep) và lặp lại nhiều bước để hoàn thành task mà không cần bạn nhắc từng bước​
C) Claude Code kết nối với internet để search Stack Overflow​
D) Claude Code không có giới hạn context window​
