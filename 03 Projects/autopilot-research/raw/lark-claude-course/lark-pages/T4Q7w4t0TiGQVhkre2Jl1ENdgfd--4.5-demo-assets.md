📎 ASSET 2 — 10 Custom Skill Ideas cho học viên​
Gợi ý theo role — học viên chọn + adapt​
💼 Role: Project Manager​
Skill idea 1: project-status-report​
•
Input: Jira board + stakeholder list​
•
Output: 1-page status report với traffic light (green/yellow/red)​
•
Use: Weekly executive update​
Skill idea 2: risk-register-generator​
•
Input: Project context + past risks​
•
Output: Structured risk register (risk, likelihood, impact, mitigation, owner)​
•
Use: Project kickoff + ongoing review​
📣 Role: Content Marketer​
Skill idea 3: content-calendar-entry​
•
Input: Topic + channel + audience​
•
Output: Calendar entry với: headline options, outline, keywords, distribution plan​
•
Use: Content planning weekly​
Skill idea 4: blog-seo-optimizer​
•
Input: Blog draft​
•
Output: SEO-optimized version + meta description + 5 keyword variants + 3 internal link suggestions​
•
Use: Pre-publish review​
