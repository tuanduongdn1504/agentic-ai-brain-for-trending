Mở đầu: Claude là intelligence, không phải một app​
Ở bài 01, chúng ta đã nói: "Claude là intelligence. claude.ai chỉ là một cách tương tác với nó."​
Giờ bài cuối của module tools, hãy unpack ý đó.​
Cùng Claude intelligence chạy ở nhiều interfaces khác nhau — mỗi cái tailored cho một loại workflow:​
•
Terminal / IDE cho developers → Claude Code​
•
Slack workspace cho team collab → Claude in Slack​
•
Microsoft Excel cho spreadsheet work → Claude for Excel​
•
Chrome browser cho web research + automation → Claude for Chrome​
Tại sao quan trọng? Vì switching tool cost real time + attention. Bạn đang code trong terminal, muốn ask Claude → open browser, tab Claude.ai, type, copy answer, paste back terminal. Each switch: 30 giây + re-focus mental overhead.​
Claude flavors specialized eliminate the switch. Claude ở đúng nơi bạn đang work.​
Bài này: quick tour của 4 specialized flavors, với tiêu chí rõ ràng khi nào pick cái nào.​
