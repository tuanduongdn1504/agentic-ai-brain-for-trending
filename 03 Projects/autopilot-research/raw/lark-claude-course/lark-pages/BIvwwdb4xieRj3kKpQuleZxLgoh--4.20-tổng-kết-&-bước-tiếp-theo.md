Hành trình 20 bài — Bạn đã đi qua đâu?​
Module 1: Nền tảng​
Bạn đã hiểu:​
•
Coding assistant khác gì autocomplete — agentic, không chỉ generate​
•
Tool use là cơ chế LM tương tác với thế giới thực​
•
Claude được train chuyên cho tool use — đó là lý do Claude Code hoạt động tốt​
Module 2: Khởi động​
Bạn đã:​
•
Cài Claude Code trên máy​
•
Authenticate với Anthropic (Pro/Max/API)​
•
Setup project thực hành (uigen hoặc project của bạn)​
Module 3: Quy trình cơ bản​
Bạn đã master:​
•
CLAUDE.md — đưa context cho Claude một lần dùng mọi request​
•
Screenshots, Planning, Thinking — 3 cách "điều khiển" Claude khi thực hiện thay đổi​
