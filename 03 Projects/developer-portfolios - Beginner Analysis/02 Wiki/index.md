## What it is

A single, massively-scaled, community-maintained **list of links** to developers' personal portfolio sites, sorted A–Z, each entry a name + URL + a few skill tags. It exists to give developers *inspiration* when building their own portfolio. Contributions arrive as PRs (fork → add your entry → PR, per `CONTRIBUTING.md`); a `feed.json` mirrors the list for programmatic consumption. ~5,977 commits over ~6.7 years reflect a long, steady contribution stream, not engineering of a system.

It is **not** a coding agent, a Claude-Code tool, an agentskills.io product, an MCP server, an LLM application, or anything in the autonomous-software-development space the corpus tracks. It contains **no AI/agent content of any kind.** It lands in this corpus only by an explicit operator override (below).
