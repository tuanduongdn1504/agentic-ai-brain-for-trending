## 5. 3 new Pattern candidates at v19

### #28 Multi-Provider AI Abstraction (LiteLLM)

**Formal statement:**
> "Agent-space frameworks increasingly adopt provider-abstraction libraries (LiteLLM, OneAPI, OpenRouter, LangChain ChatModels) to decouple framework from specific AI vendor."

**Evidence at v19:** 1 data point (TrendRadar)
**Required for promotion:** 2+ more frameworks
**Prediction:** Strong — widespread outside corpus. Likely promotes v20-v21.

### #29 GPL-3.0 in Agent-Space

**Formal statement:**
> "Copyleft licensing (GPL-3.0, AGPL) emerges as community-preservation commitment in agent-space."

**Evidence at v19:** 1 data point (TrendRadar first GPL-3.0)
**Corpus license distribution:**
- MIT: 16 wikis
- Apache-2.0: 1 (gws)
- ISC: 1 (codymaster)
- **GPL-3.0: 1 (TrendRadar)**

**Prediction:** May remain rare like Pattern #14 (alignment-theory). Permissive license dominates.

### #30 3-Layer Agent Ecosystem Stratification

**Formal statement:**
> "As agent ecosystem matures, 3 distinct standardization layers emerge:
> 1. Transport (MCP) — universal
> 2. Provider abstraction (LiteLLM-style) — emerging standard
> 3. Runtime identifiers (OpenClaw/Hermes) — community-platform + regional-specific
> 
> Frameworks adopt layers independently."

**Evidence at v19:** TrendRadar exemplifies L1 + L2 + skips L3
**Required for promotion:** 2+ more frameworks exemplifying layered adoption
**Prediction:** Strong — likely promotes v20+.

## 8. Emergent patterns at T4 4-way

### Pattern: Regional-ecosystem parallel infrastructure

TrendRadar reveals CN-ecosystem has parallel infrastructure at every ecosystem layer (runtime, LLM provider, desktop client, push channels, trending platforms). This isn't corpus-observable across Western entrants alone — need regional diversity.

### Pattern: GitHub Actions as primary agent-framework deployment

TrendRadar = first T4 to use GH Actions cron as primary production deployment. Novel pattern — scheduled agent frameworks via shared CI infrastructure.

### Pattern: MCP Resources underutilized in corpus

Most corpus frameworks expose MCP Tools only. TrendRadar's 4 MCP Resources (platforms, rss-feeds, dates, keywords) is corpus-first explicit usage. Suggests MCP spec underutilized by most frameworks.

### Pattern: Dual Docker image separation

TrendRadar separates crawler + MCP into independent Docker images. Could become convention as agent-integration matures.
